Run Linux binary server

1. Download latest binary for Linux: https://github.com/manasky/lorem-server/releases
2. Extract file
3. Open Terminal and navigate to the extracted folder. 4. Run the app (also grant permissions if needed): 
./lorem-server --host "127.0.0.1:8080" --dir "~/Pictures"
This will run the server on port 8080 and serves images from ~/Pictures directory
